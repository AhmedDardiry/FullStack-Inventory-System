# Full-Stack Inventory Management System

## 📌 Project Overview
This is a comprehensive web-based Inventory Management System designed to track product stock, manage orders, and streamline warehouse operations. The application provides a seamless connection between a dynamic user interface and a robust relational database, ensuring real-time data accuracy and efficient inventory control.

## 🛠️ Tech Stack & Technologies Used
* **Frontend:** React.js
* **Backend:** Node.js, Express.js
* **Database:** MySQL
* **Architecture:** RESTful API Design

## ⚙️ Key Features & Workflow
1. **Dynamic Dashboard:** A responsive UI built with React to display real-time inventory metrics, low-stock alerts, and product summaries.
2. **Product Management:** Full CRUD (Create, Read, Update, Delete) operations for adding new items, updating stock levels, and categorizing products.
3. **Database Integration:** Structured relational database using MySQL to securely store product details, transactions, and supplier information.
4. **API Communication:** Seamless data flow between the React frontend and MySQL database via custom-built RESTful APIs using Node.js.

## 🚀 Purpose
This project demonstrates proficiency in Full-Stack Web Development, specifically showcasing the ability to build scalable frontends with React, construct reliable backend servers with Node.js, and design efficient relational database schemas using MySQL.
